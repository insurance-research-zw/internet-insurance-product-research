# 互联网保险产品研究知识库
> 专注蚂蚁保、微信微保、京东保险线上保险产品体系深度研究
## 个人专业资质支撑
- 法律职业资格证：保险条款合规、《互联网保险业务监管办法》监管政策解读、线上宣传法律风险研判
- 保险核保核赔中级证书：智能核保分层规则、理赔风控体系、线上纠纷处理标准
- 高级健康管理师：慢病人群风险分级、医疗险配套体检/门诊/慢病管理增值服务设计
## 仓库内容目录导航
- [01-竞品拆解库](./01-竞品拆解库)：全平台短期意外险、百万医疗、长期重疾、电商履约险单品拆解、横向对比分析文档
- [02-合规风控体系](./02-合规风控体系)：产品上线合规自查SOP、条款法律风险识别清单、线上智能核保分层规则框架
- [03-健康险产品设计](./03-健康险产品设计)：非标体慢病医疗险方案、分人群健康保障分层模型、医疗增值服务组合模板
- [04-产品工具模板](./04-产品工具模板)：线上保险简化PRD模板、保费赔付数据复盘Excel模板、渠道产品合作需求方案
## 项目说明
1. 仅做行业中立客观产品研究，不售卖保险、不提供投保咨询，无商业引流行为；
2. 全部分析素材均来自平台公开页面、官方监管法规、公开司法判例，无企业内部涉密数据；
3. 完整复刻大厂保险产品专家「用户调研→竞品分析→合规校验→产品迭代」全工作链路，作为求职专业作品集；
