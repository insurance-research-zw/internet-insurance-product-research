# 线上保险产品上线合规自查SOP
## 1. 总则
适用范围：蚂蚁保/微保/京东保险线上医疗险、重疾、意外险、电商履约险、团险；覆盖产品立项→条款审核→页面设计→上线试运行→常态化巡检全流程。
依据法规：《保险法》《互联网保险业务监管办法》《金融产品网络营销管理办法》《个人信息保护法》《人身保险产品负面清单》。
自查角色：产品、法务、核保、运营、渠道联合校验，本SOP为标准化检查清单。

## 2. 阶段1：产品立项合规自查（立项前）
1. 持牌主体校验
- 投保跳转页面是否为保险公司自营平台；第三方平台仅做引流，不得截留投保流程
- 渠道合作协议是否明确权责、宣传审核主体、保费结算、可回溯责任
2. 险种经营范围校验
- 不得超出保险公司许可证载明业务范围；线上长期人身险、短期险分开备案
3. 客群与地域限制
- 互联网人身险异地展业规则校验；中老年、未成年人投保限额、监护人授权规则

## 3. 阶段2：保险条款法律合规自查（法务/产品双审）
1. 格式条款提示义务（核心）
- 免责、免赔、等待期、续保限制、减保、现金价值、既往症除外是否加粗高亮，设置强制阅读确认步骤，禁止一键全选同意
- 概括性健康告知禁止（如“其他未提及疾病”），必须逐项列明具体疾病询问事项
2. 保险责任与给付条件
- 重疾疾病定义需贴合行业统一规范；不得增设额外生存期限变相限制赔付
- 续保规则清晰写明：保证续保年限、停售是否可无等待期续保、费率可调规则公示
3. 现金价值、退保、犹豫期
- 犹豫期时长、退保扣费规则、现金价值表完整公示，无模糊表述
4. 特别约定/附加条款
- 特别约定不得与主条款冲突，不得隐性免除保险公司法定责任

## 4. 阶段3：线上投保页面&营销宣传合规自查（高频处罚点）
1. 页面强制标识
- 首页显著标注：保险产品全称、承保保险公司、销售平台名称、备案号
2. 宣传话术红线校验（逐项排查）
- 禁止“首月1元、免费领、保本、100%赔付、不限病、无免赔”等误导表述
- 不得对比贬低同业产品；赔付案例需真实可溯源，禁止虚构理赔
3. 健康告知页面合规
- 禁止折叠、缩小健康告知；病史询问分层清晰，无模糊概括提问
- 未如实告知法律后果弹窗提示，单独勾选确认框（不能与条款合并勾选）
4. 个人信息收集合规
- 仅采集承保、核保必需信息；健康、病历等敏感信息需单独授权；符合等保三级数据存储要求

## 5. 阶段4：核保、理赔、服务流程合规
1. 智能核保规则
- 核保结论（标体/除外/加费/拒保）逻辑可解释，不得设置无依据拒保条件
- 非标体延期、除外承保需书面说明依据
2. 线上理赔规则
- 理赔材料、时效、医院范围清晰公示；不得设置隐形门槛
3. 可回溯管理
- 全流程投保页面、勾选记录、弹窗提示留存≥5年电子档案，满足监管调阅要求

## 6. 阶段5：上线后常态化月度巡检SOP
1. 页面文字迭代二次审核；渠道短视频、图文宣传每周抽查
2. 客诉、理赔纠纷复盘，定位条款/页面合规漏洞
3. 监管新规同步更新自查清单，季度全产品复核

## 7. 自查输出物
- 《产品合规自查记录表》（含校验项、风险等级、整改要求、复核人、完成时限）
- 高风险项红标清单，附法律依据与整改方案
